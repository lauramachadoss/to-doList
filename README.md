# 💖 To-do List

Esse é um projeto de lista de tarefas que eu fiz para praticar desenvolvimento web do zero 🌸
A ideia foi aprender na prática como transformar um layout em código e adicionar interatividade com JavaScript.

---

## ✨ O que dá pra fazer

* Adicionar tarefas
* Colocar data e hora
* Marcar como concluída (com efeito riscado)
* Remover tarefas
* Interface simples e organizada

---

## 🛠️ Tecnologias

Usei só o básico mesmo, pra aprender bem:

* HTML
* CSS
* JavaScript

---

## 🧠 O que eu aprendi com esse projeto

* Como estruturar um layout a partir do Figma
* Usar Flexbox pra organizar os elementos
* Manipular o DOM com JavaScript
* Criar elementos dinamicamente (`createElement`)
* Trabalhar com eventos (`click`, `change`)
* Resolver erros na prática (principalmente no JS 😅)

---

## 💡 Sobre

Fiz esse projeto com foco em aprendizado mesmo, tentando entender cada etapa ao invés de só copiar código.
Foi muito útil pra conectar HTML + CSS + JS de verdade.
